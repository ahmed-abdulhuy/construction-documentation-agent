# Documents Domain Rules
Draft <br>
  ├─ can edit <br>
  ├─ can upload version <br>
  └─ can submit <br>

Submitted <br>
  ├─ locked <br>
  ├─ waiting approvals <br>
  └─ can comment <br>

Approved <br>
  ├─ immutable <br>
  ├─ downloadable <br>
  └─ legally final <br>

Rejected <br>
  ├─ read-only <br>
  └─ requires new version <br>
